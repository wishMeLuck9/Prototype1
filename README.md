# Prototype1

Unity prototype project for a driving lesson scene.

## Project Info

- Engine: Unity `6000.3.9f1`
- Main scene: `Assets/Scenes/Prototype 1.unity`
- Scripts: `Assets/Scripts/PlayerController.cs`, `Assets/Scripts/FollowPlayer.cs`

## Gameplay

The player controls a car that moves forward/backward and turns left/right.
The camera follows the car with a fixed offset.

## Controls

- `W` / `Up Arrow`: move forward
- `S` / `Down Arrow`: move backward
- `A` / `Left Arrow`: turn left
- `D` / `Right Arrow`: turn right
